Block 7 Hypothesis and Evidence

Block 7 is where the course moves from describing what the data contains into testing whether an observed difference is likely to be real or likely to be chance. The goal is not to turn statistics into a black box. The goal is to understand the order of reasoning: frame the question, state the null and alternative hypotheses, run the test, then interpret the result carefully.

NOTE 01

What This Block Adds

Plain Englishfrom description to evidence

Blocks 1 through 6 mostly ask: what is in the table, how is it shaped, what stands out, and how do two groups compare descriptively?

Block 7 adds a new question: if two groups look different, is that difference strong enough to count as evidence against the idea that nothing real is going on?

Block 7 does not replace earlier summary work. It adds a way to test it.
Block 7 sequencereasoning order

Step 1: name the two groups and one numeric measure.

Step 2: state the null hypothesis and the alternative hypothesis in plain English.

Step 3: run the test.

Step 4: interpret the p-value and then ask whether the difference matters in practice.

Main rulehypothesis before p-value

The null and alternative hypotheses must be stated before any p-value appears. The test result does not invent the question afterward. The question comes first.

No p-value belongs on the page before the hypothesis does.

NOTE 03

How The Logic Works

Hypothesistwo statements, not one

A hypothesis test always starts with two competing statements.

The null hypothesis says there is no meaningful difference or effect. The alternative hypothesis says there is a meaningful difference or effect.

Code breakdownt-test core line
stats.ttest_ind(group_a, group_b)
What the test is askinginside the black box

The t-test is not asking whether the two group means are numerically identical. It is asking whether the distance between them looks large enough, relative to the spread of the data, that the no-difference explanation starts to look weak.

A test is comparing signal against noise, not just mean against mean.
What a p-value isevidence, not verdict

A p-value measures how surprising the observed result would be if the null hypothesis were true. A smaller p-value means the result would be less expected under the no-difference story.

A larger p-value means the result is more compatible with the idea that the apparent difference could have arisen without a real underlying effect.

Rule0.05 is a convention

The threshold 0.05 is common, but it is not a law of nature. It is a convention used to organise judgment. It should never be mistaken for a magic border between truth and falsehood.

A p-value helps you reason. It does not excuse you from reasoning.
NOTE 04

Significance And Importance

Two different questionsstatistical vs practical
Question What it asks
Statistical significance Is the observed result unlikely to be explained by chance alone under the null hypothesis?
Practical importance Is the size of the difference large enough to matter in the real world?
A result can be statistically significant and still be too small to matter.
Why the means still matterdo not skip effect size

The p-value is not enough on its own. You still need to inspect the group means and the gap between them. If the difference is tiny, the real-world meaning may still be weak even when the test result looks strong.

Main cautiondo not let the test do all the thinking

Block 7 is not about handing judgment over to scipy. The test result is one piece of evidence. It must be read alongside the size of the difference, the context of the question, and the limits of what the dataset can actually support.

Statistical significance is part of the answer. It is never the whole answer.
