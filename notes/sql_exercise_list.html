NOTE 02

When a Query Fails

First response - read the error message before changing anything

At this level, SQL errors are almost always one of four things.

Common failureWhat to check
Wrong column nameCheck the exact spelling against the table shown.
Missing single quote around a text valueText values need single quotes. Numbers do not.
Wrong table nameRead the SQLZoo page carefully and check the table heading.
GROUP BY mismatchCheck whether a SELECT column is neither grouped nor aggregated.
Fix the actual error, rerun, move on. Do not thrash around randomly rewriting the whole query.
