Reference Guide

Block 5 Visualisation

Block 5 turns table results into views that are quicker to scan. The goal is not decoration. The goal is to choose a chart shape that matches the question, then state clearly what the chart shows and what it still cannot tell from that view alone.

NOTE 01

Why Use A Chart

Plain English - pattern first

A chart helps the eye notice pattern, spread, imbalance, or relationship faster than a long printed table.

A chart is not automatically an explanation. It is a view of the data from one angle.

A chart is a fast view, not the final conclusion.
NOTE 02

At-a-glance Command Boxes

Visualisation - Block 5 core moves
df["column"].value_counts().plot(kind="bar") Bar chart for category counts.
df["column"].plot(kind="hist") Histogram for one numeric column.
df.plot.scatter(x="col1", y="col2") Scatter plot for the relationship between two numeric columns.
plt - a second import alias

plt is the short nickname given to matplotlib, a separate Python library used for drawing charts. Before plt.show() can be used, matplotlib must be imported with: import matplotlib.pyplot as plt. This line works exactly like import pandas as pd — it loads the library and gives it a nickname. plt.show() tells matplotlib to display the chart that has just been built.

plt.show() Display the chart in the notebook output.
NOTE 03

Chart Choice Rule

Bar chart - category comparison

Use a bar chart when the question is about categories and counts or category totals.

Example question: which weather label appears most often?

Code breakdown - bar chart
df["weather"].value_counts().plot(kind="bar")
Histogram - one numeric column

Use a histogram when the question is about spread or distribution in one numeric column.

Example question: are most fares low, medium, or high?

Code breakdown - histogram
df["amount"].plot(kind="hist")
Scatter plot - two numeric columns

Use a scatter plot when the question is about whether two numeric columns move together or apart.

Example question: do higher score values tend to sit beside lower result values?

Code breakdown - scatter plot
df.plot.scatter(x="score", y="result")
Do not force it - wrong chart, wrong message

A chart type is not neutral. A bad chart choice makes the pattern harder to read or suggests the wrong kind of comparison.

NOTE 04

What A Chart Shows And Cannot Show

Required sentence pair - show plus limit

For every chart, write one sentence saying what the chart shows and one sentence saying what the chart cannot tell from that view alone.

A bar chart can show which category is largest. It cannot explain why that category is largest.
Useful stems - reader-facing language

Shows: This chart shows that ___ appears most often.

Shows: This histogram suggests that most values sit in the ___ range.

Shows: This scatter plot suggests a possible relationship between ___ and ___.

Cannot show: This chart does not explain why the pattern exists.

Cannot show: This view alone does not prove cause and effect.

Display step - show the chart

plt.show() is the plain instruction that tells the notebook to display the current chart output.
