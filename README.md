# Learn Python Data

An explanation-first route from core Python decisions and records into pandas,
data analysis, and a practical Python-to-SQL bridge.

Start with [GETTING_STARTED.md](GETTING_STARTED.md). Then read the bookshelf
(`index.html`) and work through each matching notebook locally.

The course deliberately separates three things:

- worked examples use small, unrelated dummy material;
- learner tasks use one clear lesson dataset or story;
- refresh checks use unfamiliar material so that recall, rather than copying,
  is doing the work.

Everything needed to run the notebooks is supplied in this repository.
