NOTE 01

What These Measures Are For

Plain Englishcentre and spread

Some statistics try to describe the centre of the data. Others try to describe the spread of the data.

No single number tells the whole story. A mean can hide skew. A mode can hide range. A standard deviation can hide shape.

A summary number is a shortcut, not the full data.

NOTE 03

What Each Measure Tries To Tell You

Meanaverage

The mean is the average. It is useful when you want one central value that takes every number into account.

It can be pulled upward or downward by unusually large or unusually small values.

Code breakdownmean
df["amount"].mean()
Medianmiddle value

The median is the middle value after sorting. It is useful when you want a centre measure that is less affected by extreme values.

Code breakdownmedian
df["amount"].median()
Modemost common value

The mode is the value that appears most often. Some columns have one mode, some have several, and some have none that are especially meaningful.

Code breakdownmode
df["category"].mode()
Standard deviationspread around the mean

Standard deviation is a spread measure. A larger value usually means the numbers are more spread out around the mean. A smaller value usually means they sit more tightly together.

It does not show shape by itself. Two datasets can share a similar standard deviation and still have very different distributions.

Code breakdownstd
df["score"].std()
Distribution shapenumbers plus picture

A histogram helps you ask whether the distribution looks balanced, skewed, clumped, or stretched. This matters because the same mean can sit on top of very different shapes.

The centre tells you one thing. The shape tells you another.
NOTE 04

Sample Vs Population

Plain English firstwhat the table stands for

A population means every case you care about. A sample means only some of those cases.

Most real-world tables in this course should be treated as samples unless you have strong evidence that the table contains every relevant case.

Why it matterscareful claims

Population claim: this number describes the whole set.

Sample claim: this number describes the table you have and may suggest something about the larger world, but it does not guarantee it.

Caution: do not quietly treat a sample summary as if it automatically describes everyone.
