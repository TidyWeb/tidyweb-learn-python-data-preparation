Seaborn Foundations

Seaborn is a Python visualisation library built on top of matplotlib. It produces publication-quality charts with less code and is used throughout the bootcamp's Module 4. This reference covers the five core chart types, when to use each one, and the commands needed to produce them.

NOTE 01

Seaborn vs Matplotlib

Plain English - what seaborn adds

matplotlib gives you full control over every element of a chart. seaborn gives you better-looking charts with less code by making sensible defaults automatic: colours, spacing, grid style, and legend placement.

seaborn does not replace matplotlib. It sits on top of it. You still call plt.show() and plt.title(). You still import matplotlib. seaborn handles the heavy visual lifting. matplotlib handles the rest.

import seaborn as sns is the only new import. Everything else stays the same.

NOTE 02

Scatter Plot

When to use it - two numeric columns, one categorical

Use a scatter plot when you want to see the relationship between two numeric columns. Add hue to colour dots by a categorical column and reveal whether the relationship differs by group.

sns.scatterplot(data=df, x="col1", y="col2")Basic scatter plot of two numeric columns.
sns.scatterplot(data=df, x="col1", y="col2", hue="cat_col")Colour dots by a categorical column to reveal group patterns.
If two numeric columns might be related, a scatter plot is the first check.
NOTE 03

Bar Plot

When to use it - compare averages across categories

Use a bar plot when you want to compare the average of a numeric column across categories. Each bar shows the mean for that group. Seaborn adds confidence interval lines by default.

sns.barplot(data=df, x="cat_col", y="num_col")One bar per category showing the mean of the numeric column.
sns.barplot(data=df, x="cat_col", y="num_col", hue="cat_col2")Split bars by a second categorical column.
Bar plots show averages. They do not show the spread of values within each group.
NOTE 04

Histogram

When to use it - distribution of one numeric column

Use a histogram when you want to see how values in one numeric column are distributed. kde=True overlays a smooth density curve that shows the shape of the distribution more clearly than the bars alone.

sns.histplot(data=df, x="num_col")Basic histogram of one numeric column.
sns.histplot(data=df, x="num_col", kde=True)Histogram with a smooth density curve overlaid.
If you want to know whether data is skewed, peaked, or spread evenly, histogram first.
NOTE 05

Box Plot

When to use it - spread and outliers by category

Use a box plot when you want to compare the spread of a numeric column across categories. The box shows the middle 50% of values. The line is the median. Dots beyond the whiskers are potential outliers.

sns.boxplot(data=df, x="cat_col", y="num_col")One box per category showing spread, median, and outliers.
sns.boxplot(data=df, x="cat_col", y="num_col", hue="cat_col2")Split boxes by a second categorical column.
A bar plot shows averages. A box plot shows spread. Use box plots when outliers matter.
NOTE 06

Heatmap

When to use it - correlation between numeric columns

Use a heatmap to visualise a correlation matrix: how strongly each numeric column relates to every other numeric column. Values near 1 indicate a strong positive relationship. Values near -1 indicate a strong negative relationship. Values near 0 indicate no meaningful relationship.

Compute the matrix with df.corr() then pass it to sns.heatmap().

sns.heatmap(df.corr(), annot=True, fmt=".2f")Heatmap of the full correlation matrix with values shown.
sns.heatmap(df[["col1","col2","col3"]].corr(), annot=True, fmt=".2f", cmap="coolwarm")Heatmap of selected columns only, with a diverging colour scale.
A heatmap of correlations is the fastest way to see which columns are related.
