## What to do when a query fails

Read the error message first. SQL errors at this level are almost always one of:

- Wrong column name - check the exact spelling against the table shown
- Missing single quote around a text value
- Wrong table name
- GROUP BY mismatch - a SELECT column is neither grouped nor aggregated

Fix the error, rerun, move on.
